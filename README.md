# Study☆Star v1.5

## 今回の更新
- ホーム画面のKAITOアドヴァンスを正式な全身画像へ差し替え
- 「学習時間を残す」にカレンダー（日付選択）を追加
- 過去の日付へさかのぼって学習時間を登録可能
- 夏休み300時間チャレンジの累計・残り・達成率を表示
- 学習記録の一覧表示と削除機能を追加
- Challenge45ポイントを正式ルールに固定
  - その日の1回目・2回目：各55P
  - その日の3回目以降：各88P
- Challenge45完了時は45分の学習時間も自動記録
- 手入力した学習時間にはChallenge45ポイントを付与しない

## 使い方
GitHub Pagesでは、このフォルダ内のファイルをすべて同じ階層にアップロードしてください。起動ファイルは `index.html` です。

## データ保存
記録はブラウザのlocalStorageへ保存されます。同じ端末・同じブラウザで利用してください。ブラウザデータを消去すると記録も消えるため、今後の版でバックアップ機能を追加予定です。
